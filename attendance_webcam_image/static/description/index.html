Attendance Webcam Image - Attendance Photo

This module helps you to capture the employees photo while check in / check and kiosk mode from Employee Attendance.

Installation

- Copy attendance_webcam_image module to addons folder

- Install the module normally like other modules

Features

  • Capture the Employees Attendance Photoe while Check in / Check out
  • Capture the Employees Attendance Photo on Kiosk Mode.

1. User Configuration

User Menu -> Attendance Webcam Image





2. Attendance

Attendances -> Check in / Check Out



Attendances -> Form View



Attendances -> Kiosk Mode.
