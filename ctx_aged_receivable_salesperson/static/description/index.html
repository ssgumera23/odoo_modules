Aged Receivable Report per salesperson

Note: This module works only in Odoo Enterprise

Overview

This module allow you to filter Aged Receivable Report by salesperson assigned to the customer in partner form view.

Features
Filter Aged Receivable report by salesperson
Show salesperson in aged receivable report.
